First-time setup
Set these Firestore Security Rules in Firebase Console → Firestore → Rules: rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } match /users/{userId}/recipes/{recipeId} { allow read: if true; } } }
